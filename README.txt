This is an android application based on this tutorial:
https://www.youtube.com/watch?v=YJ0JQXcNNTA&list=PLgCYzUzKIBE8KHMzpp6JITZ2JxTgWqDH2&index=4

What it does:
With GUI, user is able to
1. Turn bluetooth on phone on and off
2. Enable phone to be discoverable by surrounding bluetooth devices
3. Discover surrounding bluetooth devices and list them out
4. Log everything.
